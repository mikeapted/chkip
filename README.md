# chkip
Command line tool to retrieve current IP

## Installation

  `npm install @mikeapted/chkip`

## Usage

    var chkip = require('@mikeapted/chkip');

    var myIp = chkip();
  
  
  Output should be `XXX.XXX.XXX.XXX`
